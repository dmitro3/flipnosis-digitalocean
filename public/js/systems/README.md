# Game Systems Modules

This folder contains the core game mechanics:

- **tube-creator.js** - Creates and styles glass tubes (lab, cyber, mech, potion)
- **coin-manager.js** - Coin logic, rotations, flip animations, landing
- **pearl-physics.js** - Pearl/particle physics simulation and color updates
- **glass-shatter.js** - Glass shattering effects and shard physics
- **power-system.js** - Power charging, accuracy calculations, sweet spots

Each module exports functions or classes that can be imported by other modules.
